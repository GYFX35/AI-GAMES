Accessing Your Analytics Data

All user, player, and revenue-related events are now being tracked and sent to Firebase Analytics. You can view and analyze this data in your Firebase project console.

How to View Your Data:

  1. Go to the Firebase Console:

    Navigate to the Firebase Console and select your project (supergame-4aa29).

  2. Open the Analytics Dashboard:

    In the left-hand menu, under the "Analytics" section, click on "Dashboard". This will give you a high-level overview of your user engagement, including active users, user retention, and more.

  3. View Custom Events:

    To see the custom events we've set up (like play_game, login, etc.), go to the "Events" tab in the Analytics section. Here you will see a list of all tracked events and their counts. You can click on any event to see more detailed reporting, including the parameters we're tracking (like game name or login method).

  4. Use DebugView for Real-time Testing:

    To see events in real-time as you test the website, use the "DebugView". You can find this at the bottom of the left-hand menu in the Analytics section. Events from your development machine should appear here within seconds, which is very useful for verifying that tracking is working correctly.

Key Events Being Tracked:

  • login: When a user logs in with Facebook or Twitch.
  • play_game: When a user clicks a "Play Now" button.
  • sponsor_game: When a user clicks a "Sponsor this game" button.
  • dropshipping_link_click: When a user clicks a "Buy Now" link on the dropshipping page.
  • affiliate_link_click: When a user clicks on an affiliate link.
