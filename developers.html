Developer API

Welcome to the Games Universe API! Our API allows you to programmatically access our platform's features, including game search, our AI assistant, and game submission.

Interactive API Documentation

Our API is built with FastAPI, which provides automatic interactive documentation (via Swagger UI). You can use this to explore the API endpoints, see the request and response models, and even try out the API directly from your browser.

To access the interactive documentation, please ensure the API server is running and then click the button below:

Open Interactive API Docs

You will need to use the test API key test-api-key to authorize your requests in the interactive documentation.
