Developer API

Welcome to the Games Universe API! Our API allows you to programmatically access our platform's features, including game search, our AI assistant, and game submission.

Getting Started

To get started, you will need an API key. You can use the following test key for now:

test-api-key

You must include your API key in the X-API-Key header of your requests.

API Endpoints

Game Search API

Endpoint: GET /api/games/search

Description: Search for games by name or category. The response now includes richer data for each game.

Query Parameters:

  • q (string, optional): The search query. If not provided, all games will be returned.

Example Request:

curl -X GET "http://localhost:8000/api/games/search?q=AI" -H "X-API-Key: test-api-key"

Example Response:

[
    {
        "name": "AI MMO Adventure",
        "category": "AI MMO Games",
        "link": "ai_mmo_games.html",
        "description": "An open-world adventure powered by a dynamic AI storyteller.",
        "rating": 4.5,
        "player_count": 12000
    }
]

AI Assistant API

Endpoint: POST /api/ai/chat

Description: Get a response from our advanced AI assistant. You can ask for game ideas, code snippets, and more.

Request Body:

{
    "message": "Give me a game idea"
}

Example Request:

curl -X POST "http://localhost:8000/api/ai/chat" -H "X-API-Key: test-api-key" -H "Content-Type: application/json" -d '{"message": "Give me a game idea"}'

Example Response:

{
    "response": "How about a puzzle game where you play as a time-traveling detective who has to solve crimes by manipulating events in the past? Each level could be a different case with unique paradoxes to resolve."
}

Game Submission API

Manage your games on the platform with the following endpoints:

  • Create Game: POST /api/games
  • Update Game: PUT /api/games/{game_name}
  • Delete Game: DELETE /api/games/{game_name}

See the API reference for more details on the request and response formats for these endpoints.
