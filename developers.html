Developer API

Welcome to the Games Universe API! Our API allows you to programmatically access our platform's features, including game search and our AI assistant.

Getting Started

To get started, you will need an API key. You can use the following test key for now:

test-api-key

You must include your API key in the X-API-Key header of your requests.

API Endpoints

Game Search API

Endpoint: GET /api/games/search

Description: Search for games by name or category.

Query Parameters:

  • q (string, optional): The search query. If not provided, all games will be returned.

Example Request:

curl -X GET "http://localhost:8000/api/games/search?q=AI" -H "X-API-Key: test-api-key"

Example Response:

[
    {
        "name": "AI MMO Adventure",
        "category": "AI MMO Games",
        "link": "ai_mmo_games.html"
    }
]

AI Assistant API

Endpoint: POST /api/ai/chat

Description: Get a response from the AI assistant.

Request Body:

{
    "message": "hello"
}

Example Request:

curl -X POST "http://localhost:8000/api/ai/chat" -H "X-API-Key: test-api-key" -H "Content-Type: application/json" -d '{"message": "hello"}'

Example Response:

{
    "response": "Hello! How can I help you today?"
}
